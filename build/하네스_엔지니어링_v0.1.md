# 마켓데스크 — LLM 하네스 엔지니어링 v0.1

> 비결정적 LLM을 제품 파이프라인에 안정적으로 끼우기 위한 설계. 갱신: 2026-06-24.
> 코드: `worker/src/providers/*`, `worker/src/prompts.ts`, `worker/src/guardrail.ts`.

## 1. 왜 "하네스"인가
LLM은 출력이 흔들린다(형식 깨짐, 환각, 군더더기). 그래서 LLM을 그대로 쓰지 않고 **앞뒤를 코드로 감싼다**: 결정적인 부분(파싱·검증·스키마)은 코드가 맡고, 추론만 LLM에 위임한다. 이 감싸개가 하네스다.

## 2. 프로바이더 라우터 (교체 가능)
`Provider` 인터페이스 하나에 구현체를 갈아끼운다. (`worker/src/providers/index.ts`)

| 프로바이더 | 호출 방식 | 비용 | 용도 |
|---|---|---|---|
| `mock` | 정규식·키워드(LLM 없음) | 0 | 로컬에서 파이프라인 전체 검증(키 불필요) |
| `claude` | 터미널 `claude -p` 직접 호출 | 사용자 구독 | 로컬/개발 실품질(키 불필요) |
| `gemini` | API(@google/genai) | 우리 | 웹 기본(저렴) |

인터페이스 3메서드: `analyze`(메타: 제목·발간일·요약·타입·산업) / `extract`(①~⑥ 구조화 분석 + 가드레일용 numbers) / `rollup`(월별 요약의 요약). 모든 구현체가 동일 JSON 스키마를 반환하고, 파싱은 공용(`parse.ts`)으로 공유한다.

## 3. 프롬프트 설계 (`prompts.ts`)
- **틀 고정**: 문서 타입(산업/기업/뉴스) 무관 공통 구조(①한줄요약 ②핵심사실 ③동인 ④리스크 ⑤관점레이어 ⑥출처)로 강제 → 출력이 항상 같은 모양.
- **문서타입별 초점**: `docTypeGuide()` 가 ②③④의 관점만 바꾼다(산업=시장구조 / 기업=실적재무 / 뉴스=5W1H). 틀은 유지, 내용 렌즈만 교체.
- **관점 레이어 동적 구성**: 사용자가 켠 렌즈(invest/job)만 프롬프트에 포함 → 불필요한 출력 억제. 취업은 직무 페르소나(`JOB_ROLE_PERSONA`) 주입.
- **출력 예시(JSON 골격) 동봉**: 모델이 채울 빈 JSON을 그대로 보여줘 형식 이탈을 줄인다.

## 4. 형식 안정화 (CLI 파서의 핵심)
LLM(특히 CLI)은 코드펜스·머리말·문자열 내 줄바꿈으로 JSON을 깬다. 두 겹으로 방어:
- **프롬프트 측 `STRICT_JSON`**: "유효한 JSON 하나만, 코드펜스·설명 금지, 문자열 안 줄바꿈 금지" 명시.
- **파서 측 `extractJson()`**: ```json 펜스 제거 → 첫 `{`~마지막 `}` 슬라이스 → `JSON.parse`. 실패 시 **줄바꿈/탭을 공백으로 치환 후 재시도**(멀티라인 string 복구). 그래도 실패면 `{}` 반환(엔트리는 빈 채로 안전 실패).

## 5. 가드레일 = 룰 + LLM 분리 (`guardrail.ts`)
- LLM이 뽑은 핵심숫자 `{label, value, page_no}` 를 **코드가 검증**한다: 해당 페이지 텍스트에 그 숫자가 실제로 있으면 `verified=true`, 없으면(환각/잘못된 인용) `false`.
- 즉 "숫자 추출"은 LLM, "출처 대조"는 룰. 환각을 제품 레벨에서 거른다.
- 롤업은 "하위 엔트리만 근거, 새 사실·숫자 생성 금지"를 프롬프트로 강제(계층 환각 누적 차단).

## 6. 비동기 큐 = 결정적 재처리
추출·롤업은 LLM이 길어서 비동기. 로컬은 DB 폴링(`parse_status='pending'` / `rollups.status='pending'`)으로 워커가 집어 처리, 운영은 SQS. **재처리는 삭제 후 재삽입**(idempotent): 엔트리/숫자/태그를 갈아끼워 중복·잔여를 없앤다.

## 7. claude CLI 통합에서 배운 함정
- **프롬프트는 stdin 으로** 전달(긴 문서가 인자 길이 제한에 안 걸리게).
- **중립 cwd(`os.tmpdir()`)에서 실행**: 프로젝트 폴더에서 `claude -p`를 띄우면 CLAUDE.md/AGENTS.md·MCP 컨텍스트를 로드해 응답에 군더더기가 섞여 JSON이 깨졌다 → 중립 디렉터리에서 호출해 순수 JSON 확보.
- **타임아웃**(기본 180s) + 종료코드 처리로 멈춤·실패를 잡는다.

## 8. 비용/한도도 하네스의 일부
LLM 호출은 곧 비용. 추출·롤업 요청마다 `consumeAnalysis()` 로 무료 한도(하루 3회) 게이팅 → 초과 시 402(Pro/BYO 안내). 비용 통제를 코드 경로에 박아둔다.

## 9. 한 줄 원칙
**결정적인 건 코드로, 추론만 LLM으로. LLM 출력은 항상 의심하고(형식·환각) 검증을 통과한 것만 저장한다.**
