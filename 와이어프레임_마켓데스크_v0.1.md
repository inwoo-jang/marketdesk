## 3. 화면별 와이어프레임

### 화면 1. 로그인
목적: 사용자별 비공개(BYO) 보장용 간단 인증.
```
┌───────────────────────────────────────────┐
│                                           │
│             [ 마켓데스크 ]                 │
│        같은 리포트, 내 관점으로            │
│                                           │
│   {이메일                              }   │
│   {비밀번호                            }   │
│                                           │
│   [        로그인        ]                │
│   ~~~이메일로 가입~~~  ~~~구글로 계속~~~   │
│                                           │
└───────────────────────────────────────────┘
```
상태: Default / Error(자격 불일치 → 입력 하단 빨강 메시지) / Loading(버튼 스피너).

---

### 화면 2. 렌즈 선택 온보딩 (복수 선택)
목적: 내 목적(렌즈)을 고른다. **복수 선택 가능**. 최초 1회, 이후 설정에서 변경.
```
┌───────────────────────────────────────────┐
│  어떤 관점으로 리포트를 보시겠어요?        │
│  (여러 개 골라도 됩니다)                   │
│                                           │
│  ┌───────────────┐  ┌───────────────┐     │
│  │ (✔) 취업       │  │ (✔) 주식투자   │     │
│  │ 산업 구조·BM   │  │ 실적·밸류·수급 │     │
│  │ 신사업 흐름    │  │ 리스크·모멘텀  │     │
│  └───────────────┘  └───────────────┘     │
│  ┌───────────────┐  ┌───────────────┐     │
│  │ ( ) 부동산·거시│  │ ( ) 산업 실무  │     │
│  │   (준비중)     │  │   (준비중)     │     │
│  └───────────────┘  └───────────────┘     │
│                                           │
│              [ 시작하기 ]                  │
└───────────────────────────────────────────┘
```
컴포넌트: 렌즈 카드(체크형). 선택 시 테두리 강조 + 체크. 0개 선택 시 [시작하기] Disabled.
상태: Default / Disabled(미선택) / 준비중 렌즈(흐림+클릭불가, Phase 2).

---

### 화면 3. 대시보드
목적: 산업별 누적 현황 한눈에 + 새 산업/업로드 진입.
```
┌──────────┬────────────────────────────────────────┐
│ 산업      │  대시보드            렌즈▼[취업·투자] ⚙ │
│ ───────  │ ───────────────────────────────────────│
│ ▸ 반도체  │  [ + 리포트 업로드 ]  [ + 산업 추가 ]   │
│ ▸ 2차전지 │                                        │
│ ▸ 자동차  │  최근 엔트리                            │
│ ▸ 방산    │  ┌──────────────────────────────────┐  │
│          │  │ 06-22 반도체 · 한투(김OO) · [취업][투자]│
│ + 추가    │  │ 한줄: HBM 쇼티지 지속...          │  │
│          │  └──────────────────────────────────┘  │
│          │  ┌──────────────────────────────────┐  │
│          │  │ 06-21 2차전지 · 삼성 · [투자]     │  │
│          │  └──────────────────────────────────┘  │
└──────────┴────────────────────────────────────────┘
```
컴포넌트: 좌측 산업 네비 / 상단 렌즈 필터(▼ 복수) / 엔트리 카드(날짜·산업·증권사·렌즈 뱃지·한줄).
상태: Empty("아직 리포트가 없어요. 첫 리포트를 올려보세요" + 업로드 CTA) / Loading(스켈레톤 카드).

### 화면 4. 산업 상세 (일/월/연 탭)
목적: 한 산업의 시간축 누적을 본다. 탭으로 일/월/연 전환, 렌즈 필터.
```
┌──────────┬────────────────────────────────────────┐
│ 산업      │  ← 반도체            렌즈▼[취업·투자]    │
│ ▸ 반도체  │ ───────────────────────────────────────│
│ ▸ 2차전지 │  [ 일별 ] [ 월별 ] [ 연별🔒 ]   [+업로드]│
│ ...      │ ───────────────────────────────────────│
│          │  ◦ 06-22  한투(김OO)   [취업][투자]      │
│          │     HBM 쇼티지 지속, CXL 신규...         │
│          │  ◦ 06-20  미래에셋     [투자]            │
│          │     DRAM 가격 반등 전망...               │
│          │  ◦ 06-18  KB           [취업]            │
│          │                                        │
└──────────┴────────────────────────────────────────┘
```
탭 동작: 일별=엔트리 목록 / 월별→화면7 / 연별=🔒 클릭 시 "데이터가 쌓이면 열려요(Phase 2)".
상태: Empty(해당 탭 데이터 없음) / 렌즈 필터 적용 시 해당 렌즈 엔트리만.

### 화면 5. 업로드 (모달)
목적: 본인 보유 PDF 업로드 + 적용할 렌즈 확인(복수).
```
┌───────────────────────────────────────────┐
│  리포트 업로드                        [✕]  │
│ ───────────────────────────────────────── │
│  ┌─────────────────────────────────────┐  │
│  │   PDF를 여기로 끌어다 놓거나        │  │
│  │        [ 파일 선택 ]                │  │
│  └─────────────────────────────────────┘  │
│  산업:  ▼ 반도체 (자동감지됨, 변경가능)    │
│  적용 렌즈:  (✔)취업   (✔)주식투자         │
│                                           │
│  ⚠ 본인이 받은 리포트만 올려주세요.        │
│     (외부 자동수집 미지원)                 │
│                                           │
│        [ 요약 시작 ]                       │
└───────────────────────────────────────────┘
```
컴포넌트: 드롭존 / 산업 드롭다운(자동감지+수정) / 렌즈 체크(복수) / 저작권 안내.
상태: Default / 파일선택됨(파일명·용량 표시) / Error(형식·용량) / Loading("파싱·요약 중..." 진행표시) / 스캔본 감지("텍스트가 없어요 → 비전 인식으로 진행").

### 화면 6. 요약 엔트리 검토 (렌즈별 섹션 · 출처 인용) ★핵심
목적: 추출 요약을 렌즈별로 확인·수정 후 저장. 핵심숫자엔 출처 칩.
```
┌────────────────────────────────────────────────┐
│  ← 06-22 반도체 · 한투(김OO)        [PDF내보내기]│
│ ────────────────────────────────────────────── │
│  렌즈:  [ 취업 ] [ 주식투자 ]   ← 렌즈 탭        │
│ ────────────────────────────────────────────── │
│  〈취업 렌즈〉                          [수정 ✎] │
│  🚀 신사업 : CXL·HBM4 진출...                   │
│  🏭 기존사업: (구조) AI수요 구조적 ↑            │
│               (단기) 레거시 DRAM 약세           │
│  🌍 해외상황: 미중 규제, TSMC capex...          │
│  🔢 핵심숫자: HBM 매출 +120% [p.4]              │
│               capex 35조 [p.7]                  │
│               파운드리 점유율 명시 없음          │
│  🎯 PM인사이트: 밸류체인 병목=후공정...         │
│ ────────────────────────────────────────────── │
│  〈주식투자 렌즈〉                       [수정 ✎]│
│  📈 수혜/리스크 · 밸류 · 모멘텀 ...             │
│  ⚖ 가정/반대시나리오 · 투자의견(보수적 해석)    │
│  ※ 투자조언 아님, 참고용                        │
│ ────────────────────────────────────────────── │
│        [ 저장 ]   [ 다시 요약 ]                 │
└────────────────────────────────────────────────┘
```
컴포넌트: 렌즈 탭(선택 렌즈만 노출) / 틀 항목 블록 / **출처 칩 [p.N]** / "명시 없음" 표기 / 인라인 편집.
상태: Loading(요약 생성중) / Error("요약 실패 → 다시 요약", 부분결과 보존) / 저장전 이탈 시 임시보존.

### 화면 7. 월별 롤업 (요약의 요약)

│ ────────────────────────────────────────────── │
│  이 달의 한 줄: HBM 쇼티지가 흐름을 지배...      │
│                                                │
│  ✅ 공통 팩트 (여러 증권사 일치)                │
│   · HBM 공급 부족 지속                          │
│   · AI 서버 수요 상향                           │
│  ⚠ 엇갈림                                      │

## 5. 인터랙션 명세
- [업로드] → 화면5 모달 → [요약 시작] → Loading → 화면6
- 화면6 [저장] → 화면4 일별 탭에 엔트리 추가(최신 위)
- 화면4 [월별] 탭 → 화면7 (해당 월 롤업, 없으면 생성 트리거)
- 연별 탭/준비중 렌즈 → 클릭 시 Phase 2 안내 토스트
- [PDF내보내기] → 화면8 모달 → [다운로드]
- 외부 URL/링크 입력 시도 → 차단 + "본인 업로드만 지원" 안내
