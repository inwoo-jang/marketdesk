# 디자인 방향 — 리포트렌즈 (ReportLens)

> 레퍼런스: Falcon 대시보드 스타일 (깔끔 + 부드러움 + 노션식 트렌디함)
> 디자인 단계(`product-workflow:design`)에서 이 토큰·원칙으로 구현.

## 무드
- 깔끔하고 가벼운 라이트 톤. 흰 카드 + 부드러운 그림자 + 넉넉한 여백.
- 노션식: 군더더기 없는 타이포, 미묘한 경계선, 둥근 모서리, 친근함.
- 데이터(숫자·뱃지)는 시원하게, 배경은 차분하게.

## 컬러 토큰
| 역할 | 값 | 용도 |
|---|---|---|
| 배경 | `#EEF1F8` ~ `#F4F6FB` | 페이지 바탕(아주 옅은 블루그레이) |
| 카드 | `#FFFFFF` | 카드/패널 |
| 프라이머리 | `#2D5BFF` (로열블루) | 핵심 액션·강조·차트 |
| 프라이머리 라이트 | `#5B8DEF` | 보조 강조 |
| 틴트(아이콘칩) | 파스텔 블루/그린/오렌지/핑크 배경 | 산업·렌즈 아이콘 칩 |
| 성공/상승 | 배경 `#E3F8EC` + 텍스트 `#16A34A` | 상승률·완료 뱃지 |
| 잉크 | `#1D2333` / 서브 `#5A6377` / 뮤트 `#8A93A8` | 본문·보조·약한 텍스트 |
| 라인 | `#E4E8F2` | 구분선·경계 |

## 형태 토큰
- 카드 radius: 16~20px / 그림자: `0 10px 30px rgba(20,30,60,.06)` (아주 부드럽게)
- 뱃지·필: pill 형태(radius 999px), 옅은 틴트 배경 + 진한 텍스트
- 아바타/아이콘: 원형, 파스텔 배경 + 머리글자/아이콘
- 큰 숫자: bold·대형 / 라벨: 작고 뮤트

## 리포트렌즈 화면 적용
- 대시보드: 산업 카드 그리드(흰 카드), 최근 엔트리 = 리스트(프로젝트 리스트 톤)
- 렌즈 뱃지: pill(취업=블루틴트, 주식=그린틴트)
- 시간뷰 탭: 세그먼트 컨트롤(선택=프라이머리)
- 출처 칩 `[p.N]`: 작은 회색 pill
- 핵심숫자: 카드 상단 대형 숫자 + 상승/하락 pill (Falcon의 Weekly Sales 카드 톤)
- 폰트: Pretendard (한글), 숫자 강조

## 비고
- 차트는 부드러운 곡선·옅은 그라데이션 면(Falcon Total Sales 톤).
- 다크모드는 Phase 2.
